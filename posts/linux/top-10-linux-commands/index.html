Top 10 Linux Commands Every Hacker starts with

September 6, 2025

Linux is an operating system like Windows we use everyday .It is based on linux kernel . Linus Trovalds is the creator of linux kernel .

Here are Following reasons why developers and security experts prefer linux

  • Open Source (can be freely dowloaded , modifed and distributed (no license shit like microsoft windows) )
  • Customizable
  • Enhanced Security
  • Performace

Here are top 10 Linux commands you can start with as a gate to linux

1) cd#

cd stands for change directory this command is used for changing directory (folder) its like moving from one directory to another

cd /home/

2) pwd#

the command pwd is used for getting the current working directory or in simple terms knowing which folder you are currently in

pwd

3) whoami#

whoami command is used to get current user

whoami

4) ls#

ls command is used to list all files

ls
  • list find hidden files
ls -al
  • list files from specific directory without visiting
ls /path/to/directory
  • sort files based on their size
ls -s

5) ps#

get any process running in background

ps
  • get process details about specific process or application
ps aux | grep firefox

6) grep#

grep command is used like a grapper . It graps data from previous command before ‘|’ so to carry out next operation

  • This below example will grep all word The entry from file.txt
cat file.txt | grep "The"

Here cat command is used to read a file content

7) htop#

Htop is terminal based lightweight system monitoring tool for linux , Think of it as task manager for linux displaying resource usage in terminal

htop

8) kill#

As the name suggests it does the work of killing process for which you just require the process or pid (process id ) of the speific process to kill it

this id can be gained from ps command we learned above

sudo kill

NOTE:

killall command can be used to killall instances of process with same name

9) rm#

rm command is used to remove (delete) a file

rm filename
  • To recursively (r) and forcefully delete a file or folder
rm -rf foldername/filenae

10) mkdir#

create new directory (folder)

mkdir folder_name

Bonus: man command is used for manual . Its is used to get usage of any command and different parameters that can be passed to a command

man command
